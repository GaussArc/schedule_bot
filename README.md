Markdown# Telegram-бот расписания Челябинского энергетического колледжа

Дипломный проект — Telegram-бот для просмотра расписания занятий ЧЭнК.

## 1. Требования

- **Операционная система**: Windows 10/11 или Linux (Ubuntu 20.04+)
- **Python**: версия 3.8 и выше
- **Доступ к внутренней сети колледжа** (через VPN)

## 2. Установка Python

1. Скачайте и установите Python с официального сайта:  
   [https://www.python.org/downloads/](https://www.python.org/downloads/)

2. При установке **обязательно поставьте галочку** «Add Python to PATH».

3. Проверьте установку:
   ```bash
   python --version
   # или
   python3 --version



3. Создание бота и получение токена

Откройте Telegram и найдите официального бота @BotFather.
Напишите команду /newbot.
Придумайте и отправьте название бота (например: CollegeHelper2025bot).
Придумайте и отправьте username бота (обязательно должен заканчиваться на bot).
BotFather пришлёт вам токен в следующем формате:text1234567890:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx

Важно для проверяющих:
Для целей демонстрации дипломного проекта параметры подключения к Telegram-боту приведены в пояснительной записке к работе «Разработка чат-бота на языке Python для просмотра расписания колледжа» (разделы «Руководство программиста» и «Публикация и продвижение приложения»).

При развёртывании проекта вне рамок дипломной работы рекомендуется использовать собственный токен Telegram-бота.


4. Настройка VPN (обязательно!)
Бот работает только с API телеграмма.

Скачайте и установите VPN-клиент, который использует обеспечивает работу телеграмма.
Подключитесь к VPN.
Убедитесь что телеграмм у вас работает.

Без активного VPN бот не сможет получить доступ к телеграмм API.

также:

Бот работает только с IP-адресов внутренней сети колледжа.

Скачайте и установите VPN-клиент, который использует колледж.
Подключитесь к VPN колледжа.
Убедитесь, что вы можете открыть сайт pronew.chenk.ru в браузере.

Без активного VPN бот не сможет получить актуальное расписание и будет работать в демо-режиме.

5. Установка и запуск бота

Склонируйте репозиторий:Bashgit clone https://github.com/GaussArc/schedule_bot.git
cd schedule_bot
Создайте виртуальное окружение:Bashpython -m venv venv
Активируйте окружение:
Windows:Bashvenv\Scripts\activate
Linux / macOS:Bashsource venv/bin/activate

Установите зависимости:Bashpip install -r requirements.txt
Запустите бота:Bashpython bot.py

Бот должен запуститься и вывести сообщение «Бот запущен!».
6. Запуск в фоновом режиме (продакшен)
Для постоянной работы на сервере используется systemd:
Bashsudo systemctl start schedule-bot
sudo systemctl enable schedule-bot
